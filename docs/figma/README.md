# Figma Exports

Drop PNG/JPG export dari Figma ke folder ini. Convention nama file:

- `home-desktop.png` `home-mobile.png`
- `work-archive-desktop.png` `work-archive-mobile.png`
- `work-single-desktop.png` `work-single-mobile.png`
- `blog-desktop.png` `blog-mobile.png`
- `blog-single-desktop.png` `blog-single-mobile.png`
- `about-desktop.png` `about-mobile.png`
- `contact-desktop.png` `contact-mobile.png`
- `components.png` (komponen, button, card, dll. — opsional)
- `style-guide.png` (palette, typography — opsional)

## Cara export dari Figma

1. Pilih frame yang mau di-export.
2. Right panel > Export > tambahkan `2x PNG`.
3. Klik tombol Export.
4. Drop file ke folder ini.

## Setelah export selesai

Beri tahu di chat. Saya analisa via tool look_at (multimodal) lalu update PRD dan Tech Doc dengan detail spesifik dari design.
